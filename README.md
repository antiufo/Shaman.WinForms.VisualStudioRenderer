# Shaman.WinForms.VisualStudioRenderer

Renders `ToolStrip`s using a Visual Studio theme.

## Usage
```csharp
using Shaman.WinForms;

VisualStudio2013ToolStripRenderer.Initialize(toolstrip1);
```
